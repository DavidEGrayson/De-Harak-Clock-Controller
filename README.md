# DigitPanelDemo
An updated demo sender for BBBDigitPanel

Assumes ledscape is installed.

By default will run plasma demo display on boot.

# Installation

    git clone https://github.com/bigjosh/DigitPanelDemo
    cd DigitPanelDemo
    chmod +x install.sh
    ./install.sh
    ./install-service.sh
